# Unidade 3 – Experiência do Usuário em Aplicações Mobile

## Apresentação da Unidade

Nesta unidade, o foco é **experiência do usuário (UX)** e **interface (UI)** para aplicações mobile. Num desenvolvimento com ênfase em dispositivos móveis, deve-se sempre melhorar a usabilidade, a clareza e a simplicidade visual do app, pensando no comportamento real do usuário no celular.

**Objetivos de aprendizagem:**

- Aplicar o conceito de mobile-first de forma prática
- Planejar fluxos de navegação simples e coerentes
- Usar feedback visual para orientar o usuário
- Tratar estados de interface: carregamento, erro e vazio
- Adotar noções básicas de acessibilidade
- Simplificar a interface para reduzir esforço do usuário
