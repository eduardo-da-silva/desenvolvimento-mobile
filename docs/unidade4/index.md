# Unidade 4 – Progressive Web Apps (PWA)

## Apresentação da Unidade

Nas unidades anteriores, você aprendeu a criar aplicações mobile com Vue.js 3, utilizando componentes, rotas e boas práticas de experiência do usuário. Agora é hora de dar o próximo passo: transformar uma aplicação web em um **Progressive Web App (PWA)**.

Um PWA permite que sua aplicação Vue.js funcione como um aplicativo instalável, com suporte a funcionamento offline e carregamento rápido — tudo isso sem precisar publicar em lojas de aplicativos.

Nesta unidade, vamos criar um novo projeto Vue.js e evoluí-lo passo a passo até se tornar um PWA funcional.

**Objetivos de aprendizagem:**

- Compreender o que define um Progressive Web App
- Diferenciar um site comum, uma SPA e um PWA
- Configurar o suporte a PWA em um projeto Vue.js 3 com Vite
- Criar e configurar o manifesto da aplicação
- Entender o papel do Service Worker no cache e no funcionamento offline
- Aplicar estratégias básicas de cache
- Testar o funcionamento offline da aplicação
- Instalar o PWA no desktop e no celular
- Gerenciar atualizações de versão com Service Worker
