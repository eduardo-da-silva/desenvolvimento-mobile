# Unidade 1 – Panorama do Desenvolvimento Mobile com Tecnologias Web

## Apresentação da Unidade

Nesta primeira unidade, vamos entender o contexto do desenvolvimento mobile moderno e como as tecnologias web se encaixam nesse cenário. Você já conhece HTML, CSS e JavaScript, e já trabalhou com Vue.js no desenvolvimento web tradicional. Agora, vamos explorar como essas mesmas tecnologias podem ser aplicadas no desenvolvimento de aplicações móveis.

**Objetivos de aprendizagem:**

- Compreender as diferentes abordagens para desenvolvimento mobile
- Identificar quando usar tecnologias web para mobile
- Reconhecer as particularidades do ambiente móvel
- Aplicar conceitos básicos de UX mobile

## Por que estudar mobile com tecnologias web?

O mercado mobile é enorme. Bilhões de pessoas usam smartphones diariamente, e a tendência é que esse número só aumente. Para um desenvolvedor, entender como criar aplicações para esse público é essencial.

Tradicionalmente, criar apps para celular significava aprender linguagens e ferramentas específicas para cada plataforma: Swift/Objective-C para iOS, Kotlin/Java para Android. Mas há uma alternativa: **usar as tecnologias web que você já conhece**.

Neste curso, vamos explorar como HTML, CSS, JavaScript e Vue.js podem ser utilizados para criar aplicações móveis completas e funcionais, com recursos como:

- Funcionar offline
- Receber notificações push
- Acessar câmera e geolocalização
- Ser instaladas na tela inicial do celular

Vamos começar entendendo o cenário geral do desenvolvimento mobile.
