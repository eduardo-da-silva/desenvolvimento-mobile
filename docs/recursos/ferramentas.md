# Ferramentas recomendadas

## Editor de código

**Visual Studio Code**  
Editor gratuito e extensível da Microsoft. Recomendado para todo o curso.  
[https://code.visualstudio.com/](https://code.visualstudio.com/)

| ----------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------- |
| [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)                                                        | Suporte a Vue.js 3 (syntax highlight, autocomplete, diagnósticos) |

## Runtime e gerenciador de pacotes

**Node.js (LTS)**  
Ambiente de execução JavaScript. Use a versão LTS (Long Term Support).  
[https://nodejs.org/](https://nodejs.org/)

**nvm (Node Version Manager)**  
Gerenciador de versões do Node.js. Permite alternar entre versões facilmente.  
[https://github.com/nvm-sh/nvm](https://github.com/nvm-sh/nvm)

**npm**  
Gerenciador de pacotes do Node.js. Já vem instalado com o Node.js.

## Ferramentas de build

**Vite**  
Ferramenta moderna de build para projetos web. Usada para criar e desenvolver projetos Vue.js 3.  
[https://vitejs.dev/](https://vitejs.dev/)

## Navegadores para teste

**Google Chrome**  
Navegador recomendado para desenvolvimento e testes. O DevTools do Chrome possui ferramentas específicas para PWA (aba Application, Lighthouse, simulação de offline).  
[https://www.google.com/chrome/](https://www.google.com/chrome/)

**Mozilla Firefox**  
Alternativa para testes de compatibilidade. Possui boas ferramentas de desenvolvimento.  
[https://www.mozilla.org/firefox/](https://www.mozilla.org/firefox/)

## Ferramentas de PWA

**Lighthouse**  
Ferramenta de auditoria integrada ao Chrome DevTools. Avalia performance, acessibilidade, boas práticas e conformidade com PWA. Acesse pelo DevTools > aba Lighthouse.

**PWA Builder**  
Ferramenta da Microsoft para validar, pontuar e empacotar PWAs.  
[https://www.pwabuilder.com/](https://www.pwabuilder.com/)

**Favicon.io**  
Gerador online de favicons e ícones para PWA a partir de texto, imagem ou emoji.  
[https://favicon.io/](https://favicon.io/)

**RealFaviconGenerator**  
Gerador completo de ícones para PWA, com preview em diferentes plataformas.  
[https://realfavicongenerator.net/](https://realfavicongenerator.net/)

## Controle de versão

**Git**  
Sistema de controle de versão distribuído. Essencial para qualquer projeto de software.  
[https://git-scm.com/](https://git-scm.com/)

**GitHub**  
Plataforma de hospedagem de repositórios Git, com funcionalidades de colaboração.  
[https://github.com/](https://github.com/)
